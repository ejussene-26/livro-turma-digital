# 📚 Livro de Turma Digital

**Sistema de Gestão Escolar Digital para Ensino Técnico Profissional e Ensino Geral**

## 🎯 Visão Geral

Uma plataforma web/mobile que centraliza a gestão de:
- ✅ Marcação de presença com geolocalização
- 📊 Assiduidade mensal de professores
- ⏰ Controle de horários e horas extras
- 👨‍💼 Supervisão em tempo real para direções provinciais
- 🔒 Prevenção de fraudes e conflitos
